# KANBAN Personalizado

Sistema de gestión de proyectos colaborativo desarrollado con Next.js, TypeScript y Tailwind CSS.

## 🚀 Tecnologías

- **Frontend**: Next.js 14 (App Router), React 18, TypeScript
- **Estilos**: Tailwind CSS
- **Base de datos**: PostgreSQL (Neon)
- **Autenticación**: NextAuth.js
- **Emails**: Resend
- **Deployment**: Vercel

## 📋 Estado del Proyecto

### Sprint 1 - Infraestructura Base
- ✅ Tarjeta 1.1: Setup Inicial
- ⏳ Tarjeta 1.2: Base de Datos (Neon + Prisma)
- ⏳ Tarjeta 1.3: Autenticación Básica

## 🛠️ Desarrollo Local

Este proyecto se desarrolla principalmente usando las interfaces web de:
- GitHub (gestión de código)
- Vercel (deployment)
- Neon (base de datos)

## 📝 Notas

Proyecto desarrollado usando metodología KANBAN para gestionar su propio desarrollo.
